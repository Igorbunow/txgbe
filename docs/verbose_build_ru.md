# Подробная сборка с сохранением временных файлов

[ English ](verbose_build.md) | [ Русский ](verbose_build_ru.md)

Чтобы выполнить максимально подробную сборку и сохранить все промежуточные результаты компиляции (например, для отладки макросов или проверки флагов), используйте следующие команды:

```bash
# Очистка предыдущей сборки
make -C /lib/modules/$(uname -r)/build M="$PWD/src" clean

# Сборка с V=1 (verbose) и сохранением временных файлов (.i, .s)
make -C /lib/modules/$(uname -r)/build M="$PWD/src" modules \
  V=1 KCFLAGS="-save-temps=obj" -j1 2>&1 | tee build_verbose_save_temps.log
```

**Что это дает:**
- `V=1`: выводит полные команды компиляции, выполняемые kbuild.
- `-save-temps=obj`: сохраняет предобработанные файлы (`.i`) и ассемблерный код (`.s`) в директории `src/`.
- `tee ...`: записывает весь вывод сборки в лог-файл для последующего анализа.
